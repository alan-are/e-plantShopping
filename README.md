# e-plantShopping
e-plantShopping 
Paradise Nursery es una tienda virtual (e-commerce) de una sola página construida con React y Redux, enfocada en la venta de plantas especializadas bajo el lema "Where Green Meets Serenity".

En resumen, la aplicación cuenta con:

Catálogo categorizado: Un menú visual que divide los productos en secciones útiles para el cliente (Plantas purificadoras de aire, Aromáticas, Repelentes de insectos, Medicinales y de Bajo mantenimiento).

Tarjetas de producto: Cada planta se muestra con su fotografía, nombre, descripción, precio y un botón interactivo.

Carrito de compras inteligente: Gracias a Redux, la tienda tiene un carrito de compras global que lleva la cuenta en tiempo real de los artículos seleccionados, cambia el estado visual de los botones cuando un producto ya fue añadido y evita que se dupliquen elementos.

Interfaz dinámica: Una barra de navegación fija que le permite al usuario cambiar fácilmente entre la vista del catálogo completo y el resumen de su carrito de compras.
